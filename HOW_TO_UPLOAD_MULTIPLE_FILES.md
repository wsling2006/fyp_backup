## ğŸ“‹ How to Select Multiple Files

### **Method 1: Hold Ctrl/Cmd While Clicking** (Recommended)

#### **On Windows/Linux:**
1. Click "Choose Files" button
2. **Hold down the `Ctrl` key**
3. Click on first file
4. **Keep holding `Ctrl`**
5. Click on second file
6. Click on third file
7. Click on as many files as you want
8. Release `Ctrl`
9. Click "Open"

#### **On Mac:**
1. Click "Choose Files" button
2. **Hold down the `Cmd` (âŒ˜) key**
3. Click on first file
4. **Keep holding `Cmd`**
5. Click on second file
6. Click on third file
7. Click on as many files as you want
8. Release `Cmd`
9. Click "Open"

---

### **Method 2: Select a Range of Files**

#### **On Windows/Linux:**
1. Click "Choose Files" button
2. Click on the **first file**
3. **Hold down the `Shift` key**
4. Click on the **last file**
5. All files between will be selected
6. Click "Open"

#### **On Mac:**
1. Click "Choose Files" button
2. Click on the **first file**
3. **Hold down the `Shift` key**
4. Click on the **last file**
5. All files between will be selected
6. Click "Open"

---

### **Method 3: Drag and Drop** (Easiest!)

1. Open your file manager (Explorer/Finder)
2. Select multiple files:
   - **Ctrl+Click** (Windows/Linux) or **Cmd+Click** (Mac) for individual files
   - **Shift+Click** for a range of files
3. **Drag** all selected files
4. **Drop** them onto the file input area in the form

## ğŸ” Troubleshooting

### **Problem: Can only select one file at a time**

**Solution 1: Make sure you're holding Ctrl/Cmd**
- âŒ Wrong: Click file 1 â†’ Release â†’ Click file 2 (only file 2 is selected)
- âœ… Correct: Click file 1 â†’ **Hold Ctrl** â†’ Click file 2 â†’ Click file 3 â†’ Release Ctrl

**Solution 2: Check the file picker window**
- Look at the bottom of the file picker dialog
- It should show: "3 files" or "5 files" etc.
- If it only shows one filename, you didn't hold Ctrl/Cmd properly

**Solution 3: Try drag and drop instead**
- Select files in your file manager first
- Then drag them all at once to the form

---

### **Problem: Files aren't showing in the list**

**Solution 1: Check the "Selected Files" section**
- After selecting files, scroll down
- You should see a gray box with all filenames listed
- If you don't see it, the files weren't selected

**Solution 2: Try clicking Choose Files again**
- Start fresh
- Hold Ctrl/Cmd from the start
- Select files one by one while holding the key

**Solution 3: Clear browser cache**
- Press Ctrl + Shift + R (Windows/Linux)
- Press Cmd + Shift + R (Mac)
- This reloads the page with latest code

### **Tip 1: Organize files first**
Before selecting:
- Put all files you want to upload in one folder
- Rename them clearly (e.g., 2026_Q1_Report.pdf)

- System will scan anyway, but this helps catch issues early
